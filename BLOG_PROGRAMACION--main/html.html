¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje que se utiliza para crear la estructura de las páginas web. Con HTML podemos agregar títulos, párrafos, imágenes, enlaces, tablas y formularios.

HTML es considerado el lenguaje principal del desarrollo web y trabaja junto con CSS para el diseño y JavaScript para la interactividad de las páginas.

Ejemplo de código HTML

<h1>Hola Angie</h1>

Características de HTML

  • Es fácil de aprender
  • Se usa para crear la estructura de las páginas web
  • Funciona en todos los navegadores
  • Permite insertar imágenes, videos y enlaces

<img> Se usa para insertar imágenes en la página

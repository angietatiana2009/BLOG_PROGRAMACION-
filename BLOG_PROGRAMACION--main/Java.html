¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web.

Con JavaScript podemos crear botones, animaciones, validaciones de formularios y muchas funciones que hacen que las páginas sean más dinámicas.

Ejemplo de código JavaScript

alert("Hola mundo");

Funciones de JavaScript

  • Crear interactividad en la página
  • Validar formularios
  • Modificar contenido HTML
  • Mostrar mensajes al usuario

Tabla de funciones JavaScript

Función Descripción
alert() Muestra un mensaje en pantalla
console.log() Muestra información en la consola
prompt() Pide información al usuario

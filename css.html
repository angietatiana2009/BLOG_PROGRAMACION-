¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje utilizado para dar estilo y diseño a las páginas web creadas con HTML. Permite modificar colores, tamaños, posiciones, fuentes y muchos otros elementos visuales.

CSS trabaja junto con HTML y JavaScript para crear páginas web modernas, atractivas y organizadas.

Ejemplo de código CSS

Un ejemplo básico de CSS sería:

body {
background-color: lightblue;
font-family: Arial;
}

Características de CSS

  • Permite cambiar colores y fondos
  • Controla tamaños de texto
  • Organiza el diseño de la página
  • Hace que las páginas se vean más profesionales

Pasos para usar CSS

  1. Crear un archivo llamado estilos.css
  2. Conectar el archivo CSS con el HTML
  3. Escribir reglas de estilo
  4. Visualizar los cambios en el navegador

Propiedades básicas de CSS

background-color Cambia el color del fondo
